# Build Bank Statement Analyzer Pro (Excel Add-in + SaaS Platform)

You are a senior full-stack architect and developer.

Build a production-quality application called **Bank Statement Analyzer Pro**.

## Objective

Create a Microsoft Excel Add-in that allows users to import bank statements, analyze spending, generate financial insights, create budgets, forecast cashflow, and export reports.

The solution must include:

1. Excel Add-in (React + Office.js)
2. Web Admin Portal (React)
3. Flask REST API Backend
4. SQLite Database
5. Authentication & Authorization
6. Subscription Billing
7. USDT Payments
8. AI Features
9. Reporting Engine
10. OCR Processing
11. Multi-bank Statement Processing

The application must be fully functional, modular, scalable, and ready for deployment.
